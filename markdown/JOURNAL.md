# Journal — Implementation Guide

Adds a voice-triggered journal system that activates on phrases like *"start journal entry"*.
When active, the UI shifts into a dedicated dictation mode: each mic press appends a new segment
to a running transcript rather than sending to the LLM. When the user finishes dictating and says
*"submit"* (or clicks a Submit button), the full raw transcript is sent to the LLM for
summarisation, the summary is shown for approval, and — on confirmation — the entry is saved to
disk by the FastAPI backend.

Entries can be retrieved by voice: *"read my last journal entry"*, *"search journal for [keyword]"*,
*"show today's entries"*.

---

## Overview

```
"Start journal entry"
        ↓
Enter journal mode — UI shifts to dictation panel
        ↓
Each mic press → Whisper STT → appended to running transcript buffer
(normal LLM pipeline is bypassed while journal mode is active)
        ↓
"Submit" (voice or button)
        ↓
Full transcript → sendToOllama() for summarisation (ephemeral)
        ↓
Summary displayed for user approval
        ↓
"Confirm" → POST /journal/entry → saved to disk
"Re-record" → buffer cleared, dictation restarts
"Discard" → exit journal mode, nothing saved
        ↓
Exit journal mode — UI returns to normal
```

**Retrieval:**
```
"Read my last journal entry"  → GET /journal/entries?limit=1 → LLM reads aloud
"Search journal for [topic]"  → GET /journal/search?q=[topic] → LLM summarises results
"Show today's entries"        → GET /journal/entries?date=today → panel lists them
```

---

## Architecture Decisions

| Decision | Choice | Reason |
|---|---|---|
| Storage format | One JSON file per entry in `memory/journal/` | Human-readable, searchable with grep, easy to back up, no database needed |
| Filename | `YYYY-MM-DD_HH-MM-SS.json` | Lexicographic sort = chronological order; collision-safe to the second |
| Summary model | Ephemeral `sendToOllama` call | Same LLM already running; no extra model needed; ephemeral so journal content doesn't contaminate conversation history |
| Approval before save | Yes — LLM summary shown first | Prevents accidental saves of garbled transcripts; user sees exactly what will be stored |
| Multi-segment dictation | Mode flag `_journalMode` gates `onstop` | Cleanest intercept: one flag checked at the top of `onstop` redirects all mic presses while active |
| Tags | Auto-generated by LLM during summarisation | Free-text keywords extracted alongside the summary in a single LLM call |
| Search | Backend full-text scan of JSON files | Simple, no vector DB needed for personal journal sizes |

### Step 3 — Create `backend/journal_routes.py`

> **Naming note:** The file is named `journal_routes.py` (not `journal.py`) to avoid any
> potential collision with stdlib modules.

```python
"""
backend/journal_routes.py
Journal storage and retrieval.
Entries are stored as individual JSON files in JOURNAL_DIR.
Exposes POST /journal/entry, GET /journal/entries, GET /journal/search.
"""

import json
import os
import re
from datetime import datetime, timezone
from pathlib import Path
from typing import Optional

from fastapi import APIRouter, HTTPException
from pydantic import BaseModel

router = APIRouter()

# ── Config ─────────────────────────────────────────────────────────────────────
_JOURNAL_DIR  = Path(os.getenv("JOURNAL_DIR", "memory/journal"))
_MAX_ENTRIES  = int(os.getenv("JOURNAL_MAX_ENTRIES", "500"))


# ── Models ─────────────────────────────────────────────────────────────────────
class JournalEntryIn(BaseModel):
    raw_transcript: str
    summary:        str
    tags:           list[str] = []
    # Client sends these so the timestamp reflects when dictation occurred,
    # not when the HTTP request arrived.
    recorded_at:    Optional[str] = None   # ISO 8601 string


class JournalEntryOut(BaseModel):
    id:             str
    recorded_at:    str
    summary:        str
    raw_transcript: str
    tags:           list[str]


# ── Helpers ────────────────────────────────────────────────────────────────────

def _entry_path(entry_id: str) -> Path:
    return _JOURNAL_DIR / f"{entry_id}.json"


def _load_entry(path: Path) -> dict:
    return json.loads(path.read_text(encoding="utf-8"))


def _all_entry_paths(newest_first: bool = True) -> list[Path]:
    """Return all .json paths in JOURNAL_DIR sorted by filename (= chronological)."""
    _JOURNAL_DIR.mkdir(parents=True, exist_ok=True)
    paths = sorted(_JOURNAL_DIR.glob("*.json"))
    return list(reversed(paths)) if newest_first else paths


# ── Endpoints ──────────────────────────────────────────────────────────────────

@router.post("/journal/entry", response_model=JournalEntryOut, status_code=201)
def save_journal_entry(body: JournalEntryIn):
    """
    Save a journal entry to disk.
    Returns the saved entry including its auto-generated id.
    """
    _JOURNAL_DIR.mkdir(parents=True, exist_ok=True)

    # Use client-supplied timestamp if valid, else server time
    try:
        dt = datetime.fromisoformat(body.recorded_at) if body.recorded_at else None
    except ValueError:
        dt = None
    if dt is None:
        dt = datetime.now(timezone.utc)

    entry_id = dt.strftime("%Y-%m-%d_%H-%M-%S")

    # Handle the (unlikely) case of a same-second collision
    path = _entry_path(entry_id)
    if path.exists():
        entry_id += f"_{int(dt.microsecond / 1000):03d}"
        path = _entry_path(entry_id)

    entry = {
        "id":             entry_id,
        "recorded_at":    dt.isoformat(),
        "summary":        body.summary.strip(),
        "raw_transcript": body.raw_transcript.strip(),
        "tags":           [t.strip().lower() for t in body.tags if t.strip()],
    }
    path.write_text(json.dumps(entry, ensure_ascii=False, indent=2), encoding="utf-8")
    return entry


@router.get("/journal/entries")
def list_journal_entries(
    limit:  int = 10,
    offset: int = 0,
    date:   Optional[str] = None,   # "today" | "YYYY-MM-DD"
):
    """
    Return recent journal entries, newest first.
    Optional ?date=today or ?date=YYYY-MM-DD filters to a specific day.
    """
    paths = _all_entry_paths(newest_first=True)

    if date:
        target = (
            datetime.now().strftime("%Y-%m-%d")
            if date.lower() == "today"
            else date
        )
        paths = [p for p in paths if p.stem.startswith(target)]

    total  = len(paths)
    sliced = paths[offset: offset + min(limit, _MAX_ENTRIES)]
    entries = []
    for p in sliced:
        try:
            entries.append(_load_entry(p))
        except Exception:
            continue

    return {"entries": entries, "total": total, "offset": offset, "limit": limit}


@router.get("/journal/search")
def search_journal(q: str, limit: int = 10):
    """
    Simple full-text search across summary + raw_transcript + tags.
    Case-insensitive substring match — no vector DB needed for personal volumes.
    """
    if not q.strip():
        raise HTTPException(status_code=400, detail="Query cannot be empty")

    query   = q.strip().lower()
    results = []

    for p in _all_entry_paths(newest_first=True):
        try:
            entry = _load_entry(p)
        except Exception:
            continue
        haystack = (
            entry.get("summary", "")        + " " +
            entry.get("raw_transcript", "") + " " +
            " ".join(entry.get("tags", []))
        ).lower()
        if query in haystack:
            results.append(entry)
        if len(results) >= limit:
            break

    return {"results": results, "total": len(results), "query": q}


@router.delete("/journal/entry/{entry_id}", status_code=204)
def delete_journal_entry(entry_id: str):
    """Delete a single journal entry by id. Used by the 'discard' flow if needed."""
    # Sanitise: only allow the filename pattern we generate
    if not re.fullmatch(r'[\d\-_]+', entry_id):
        raise HTTPException(status_code=400, detail="Invalid entry id")
    path = _entry_path(entry_id)
    if not path.exists():
        raise HTTPException(status_code=404, detail="Entry not found")
    path.unlink()
```

// frontend/journal-panel.js
// Journal dictation mode, review flow, save, and read-back.
// All journal-specific logic lives here; app.js only needs to call
// the exported functions and check _journalMode.

const BACKEND_BASE_JNL = 'http://localhost:8000';

// ── DOM refs ──────────────────────────────────────────────────────────────────
const journalPanel        = document.getElementById('journal-panel');
const journalTimestamp    = document.getElementById('journal-timestamp');
const journalTranscript   = document.getElementById('journal-transcript');
const journalSegCount     = document.getElementById('journal-segment-count');
const journalSubmitBtn    = document.getElementById('journal-submit-btn');
const journalDiscardBtn   = document.getElementById('journal-discard-btn');
const journalSummaryEl    = document.getElementById('journal-summary');
const journalTagsRow      = document.getElementById('journal-tags-row');
const journalConfirmBtn   = document.getElementById('journal-confirm-btn');
const journalRerecordBtn  = document.getElementById('journal-rerecord-btn');
const journalDiscardBtn2  = document.getElementById('journal-discard-btn-2');
const journalEntriesView  = document.getElementById('journal-entries-view');
const journalEntriesList  = document.getElementById('journal-entries-list');
const journalEntriesTitle = document.getElementById('journal-entries-title');
const journalEntriesClose = document.getElementById('journal-entries-close');

const vDictation = document.getElementById('journal-dictation-view');
const vReview    = document.getElementById('journal-review-view');
const vEntries   = document.getElementById('journal-entries-view');

// ── Mode state — exported so app.js can check it ──────────────────────────────
export let journalMode = false;   // true while dictating or reviewing

// ── Internal state ────────────────────────────────────────────────────────────
let _segments     = [];    // array of transcript strings from each mic press
let _startedAt    = null;  // ISO timestamp of when dictation began
let _pendingSave  = null;  // { summary, tags, raw } awaiting user confirmation

// ── Trigger detection ─────────────────────────────────────────────────────────

/**
 * Detect a journal start trigger.
 * Activation phrases:
 *   "start journal entry"
 *   "new journal entry"
 *   "open journal"
 *   "add journal entry"
 *   "journal entry"
 */
export function detectJournalStartTrigger(transcript) {
  const t = transcript.trim().toLowerCase();
  const patterns = [
    /\b(?:start|begin|new|open|add|create)\b.{0,20}\bjournal\b/,
    /\bjournal\s+(?:entry|note|log)\b/,
    /\bstart\s+(?:a\s+)?(?:new\s+)?entry\b/,
  ];
  return patterns.some(p => p.test(t)) ? true : null;
}

/**
 * Detect a journal submit phrase (spoken while in journal dictation mode).
 * Activation phrases:
 *   "submit"
 *   "done"
 *   "finished"
 *   "save entry"
 *   "that's all"
 *   "submit entry"
 */
export function detectJournalSubmit(transcript) {
  const t = transcript.trim().toLowerCase();
  const patterns = [
    /^\s*(?:submit|done|finished|complete|end entry|that(?:'s| is) all|submit entry|save entry|save it)\s*$/,
    /\b(?:submit|save)\s+(?:the\s+)?(?:entry|journal)\b/,
  ];
  return patterns.some(p => p.test(t)) ? true : null;
}

/**
 * Detect a journal review/retrieval trigger.
 * Activation phrases:
 *   "read my last journal entry"
 *   "show journal"
 *   "search journal for X"
 *   "what did I write about [topic]"
 *   "show today's entries"
 */
export function detectJournalReadTrigger(transcript) {
  const t = transcript.trim().toLowerCase();

  // Search
  const searchMatch = t.match(/\b(?:search|find|look up|look for)\b.{0,15}\bjournal\b.{0,10}\bfor\s+(.+)/);
  if (searchMatch) return { action: 'search', query: searchMatch[1].trim() };

  // "what did I write about X"
  const writeMatch = t.match(/\bwhat\s+did\s+i\s+(?:write|say|note)\b.{0,20}\babout\s+(.+)/);
  if (writeMatch) return { action: 'search', query: writeMatch[1].trim() };

  // Today's entries
  if (/\btoday(?:'s)?\s+(?:journal\s+)?entries\b/.test(t) ||
      /\bwhat\s+(?:did\s+i\s+)?(?:write|log)\s+today\b/.test(t)) {
    return { action: 'today' };
  }

  // Generic list / read-back
  const listPatterns = [
    /\b(?:show|open|view|read|display)\b.{0,20}\b(?:my\s+)?journal\b/,
    /\b(?:read|show)\s+(?:my\s+)?(?:last|latest|recent)\s+journal\b/,
    /\bjournal\s+(?:entries|history|log)\b/,
  ];
  if (listPatterns.some(p => p.test(t))) return { action: 'list' };

  return null;
}

// ── Dictation mode ─────────────────────────────────────────────────────────────

/**
 * Enter journal dictation mode. Called from app.js on trigger detection.
 * Accepts the trigger transcript so it can be appended to chat.
 */
export function enterJournalMode() {
  _segments  = [];
  _startedAt = new Date().toISOString();
  journalMode = true;

  // Show dictation view, hide others
  _showView('dictation');

  journalTranscript.innerHTML =
    '<span class="journal-placeholder">Start speaking. Each mic press adds a new segment. ' +
    'Press SUBMIT or say "submit" when finished.</span>';
  journalSegCount.textContent = '0 segments';

  const local = new Date(_startedAt).toLocaleString('en-US', {
    weekday: 'short', month: 'short', day: 'numeric',
    hour: '2-digit', minute: '2-digit',
  });
  journalTimestamp.textContent = local.toUpperCase();

  journalPanel.classList.remove('hidden');
  journalPanel.scrollIntoView({ behavior: 'smooth', block: 'nearest' });

  // Add CSS class to starling container for accent border
  document.getElementById('starling')?.classList.add('journal-mode');
}

/**
 * Append a new dictated segment to the running buffer.
 * Called from app.js onstop when journalMode is true and no submit phrase detected.
 */
export function appendJournalSegment(transcript) {
  _segments.push(transcript.trim());
  // Re-render transcript — each segment on its own line with a bullet
  journalTranscript.innerHTML = '';
  _segments.forEach((seg, i) => {
    const line = document.createElement('div');
    line.style.cssText = 'margin-bottom:6px;';
    line.innerHTML =
      `<span style="color:#444;font-size:0.65em;margin-right:6px;">${String(i + 1).padStart(2,'0')}</span>` +
      `<span>${_esc(seg)}</span>`;
    journalTranscript.appendChild(line);
  });
  journalTranscript.scrollTop = journalTranscript.scrollHeight;
  journalSegCount.textContent =
    `${_segments.length} segment${_segments.length !== 1 ? 's' : ''}`;
}

// ── Submit ─────────────────────────────────────────────────────────────────────

/**
 * Called when the user says "submit" or clicks the SUBMIT button.
 * Sends the full transcript to the LLM for summarisation and shows the review view.
 * Requires access to app.js's sendToOllama and SYSTEM_PROMPT — these are passed in.
 */
export async function submitJournalEntry(sendToOllamaFn, systemPrompt) {
  if (!_segments.length) {
    _speak('No content to save. Please dictate something first.', sendToOllamaFn);
    return;
  }

  const rawTranscript = _segments.join('\n\n');
  const now = new Date();
  const dateLine = now.toLocaleDateString('en-US', {
    weekday: 'long', year: 'numeric', month: 'long', day: 'numeric',
  });
  const timeLine = now.toLocaleTimeString('en-US', { hour: '2-digit', minute: '2-digit' });

  // Show a thinking state in the panel while the LLM runs
  _showView('review');
  journalSummaryEl.textContent = 'Summarising…';
  journalTagsRow.innerHTML = '';

  const prompt =
    `The following is a personal journal entry dictated by voice on ${dateLine} at ${timeLine}. ` +
    `Please write a concise third-person summary of the entry in two to four sentences — ` +
    `capturing the key thoughts, feelings, events, and decisions mentioned. ` +
    `Then, on a new line beginning with "TAGS:", list three to six single-word or short-phrase ` +
    `tags that describe the topics covered (e.g. "TAGS: work, frustration, goals, project"). ` +
    `Do not add any other commentary.\n\n` +
    `JOURNAL ENTRY:\n${rawTranscript}`;

  let summaryRaw = '';
  try {
    summaryRaw = await sendToOllamaFn(prompt, {
      ephemeralMessages: [{ role: 'system', content: systemPrompt }],
    }) ?? '';
  } catch { summaryRaw = ''; }

  // Parse summary and tags out of the LLM response
  const tagLineMatch = summaryRaw.match(/TAGS?:\s*(.+)/i);
  const tags = tagLineMatch
    ? tagLineMatch[1].split(',').map(t => t.trim().toLowerCase()).filter(Boolean)
    : [];
  const summary = summaryRaw.replace(/TAGS?:.*/is, '').trim();

  _pendingSave = { summary, tags, rawTranscript };

  // Render review
  journalSummaryEl.textContent = summary || '(No summary generated — you can still save the raw transcript.)';
  journalTagsRow.innerHTML = '';
  tags.forEach(tag => {
    const span = document.createElement('span');
    span.className   = 'journal-tag';
    span.textContent = tag;
    journalTagsRow.appendChild(span);
  });
}

// ── Confirm / Re-record / Discard ─────────────────────────────────────────────

export async function confirmJournalEntry() {
  if (!_pendingSave) return null;
  const { summary, tags, rawTranscript } = _pendingSave;

  try {
    const res = await fetch(`${BACKEND_BASE_JNL}/journal/entry`, {
      method:  'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        raw_transcript: rawTranscript,
        summary,
        tags,
        recorded_at: _startedAt,
      }),
    });
    if (!res.ok) throw new Error(`Save failed: ${res.status}`);
    const saved = await res.json();
    exitJournalMode();
    return saved;
  } catch (err) {
    console.error('[journal] save failed:', err);
    return null;
  }
}

export function rerecordJournalEntry() {
  _segments    = [];
  _pendingSave = null;
  _showView('dictation');
  journalTranscript.innerHTML =
    '<span class="journal-placeholder">Re-recording. Speak new segments and press SUBMIT when done.</span>';
  journalSegCount.textContent = '0 segments';
}

export function exitJournalMode() {
  journalMode = false;
  _segments   = [];
  _pendingSave = null;
  journalPanel.classList.add('hidden');
  document.getElementById('starling')?.classList.remove('journal-mode');
}

// ── Read-back / retrieval ──────────────────────────────────────────────────────

export async function handleJournalRead(trigger, sendToOllamaFn, systemPrompt) {
  let entries = [];
  let titleLabel = 'RECENT ENTRIES';

  try {
    if (trigger.action === 'search') {
      const res  = await fetch(
        `${BACKEND_BASE_JNL}/journal/search?q=${encodeURIComponent(trigger.query)}&limit=10`
      );
      const data = await res.json();
      entries    = data.results ?? [];
      titleLabel = `SEARCH: ${trigger.query.toUpperCase()}`;
    } else if (trigger.action === 'today') {
      const res  = await fetch(`${BACKEND_BASE_JNL}/journal/entries?date=today&limit=20`);
      const data = await res.json();
      entries    = data.entries ?? [];
      titleLabel = "TODAY'S ENTRIES";
    } else {
      const res  = await fetch(`${BACKEND_BASE_JNL}/journal/entries?limit=10`);
      const data = await res.json();
      entries    = data.entries ?? [];
    }
  } catch (err) {
    console.error('[journal] read failed:', err);
    return null;
  }

  // Show the entries panel
  _showView('entries');
  journalEntriesTitle.textContent = titleLabel;
  journalPanel.classList.remove('hidden');
  journalEntriesList.innerHTML = '';

  if (!entries.length) {
    journalEntriesList.innerHTML =
      '<div style="font-size:0.7rem;color:#444;padding:4px 0;">No entries found.</div>';
  } else {
    entries.forEach(e => {
      const card = document.createElement('div');
      card.className = 'journal-entry-card';
      const dt = new Date(e.recorded_at).toLocaleString('en-US', {
        weekday: 'short', month: 'short', day: 'numeric',
        hour: '2-digit', minute: '2-digit',
      });
      card.innerHTML = `
        <div class="journal-entry-date">${_esc(dt.toUpperCase())}</div>
        <div class="journal-entry-summary">${_esc(e.summary)}</div>
        <div class="journal-entry-tags">
          ${(e.tags || []).map(t => `<span class="journal-tag">${_esc(t)}</span>`).join('')}
        </div>
      `;
      journalEntriesList.appendChild(card);
    });
  }

  journalPanel.scrollIntoView({ behavior: 'smooth', block: 'nearest' });

  // Build LLM context and speak a summary
  if (!entries.length) {
    return 'No journal entries found for that query.';
  }
  const count   = entries.length;
  const context = entries.slice(0, 5).map((e, i) =>
    `Entry ${i + 1} (${new Date(e.recorded_at).toLocaleDateString('en-US', { weekday: 'long', month: 'long', day: 'numeric' })}):\n${e.summary}`
  ).join('\n\n');

  return `[JOURNAL ENTRIES — ${count} found]\n\n${context}`;
}

## Step 9 — Recommended Intercept Order

With journal added, the full ordering in both `onstop` and `handleSend` is:

```
1. journalMode active check       — MUST be first; redirects all mic activity
2. _matchesExitPhrase             — dossier exit
3. _parseTrigger                  — dossier open
4. detectJournalStartTrigger      — journal start
5. detectJournalReadTrigger       — journal read-back / search
6. detectTimerTrigger             — timer
7. detectTimeTrigger              — time query
8. detectWeatherTrigger           — weather
9. detectCalendarTrigger          — calendar
10. detectNewsTrigger             — news
11. detectMarketTrigger           — stocks/crypto
12. appendMessage + sendToOllama  — normal LLM path
```

The `journalMode` check at position 1 is critical — it must intercept before any other trigger
can fire and consume the "submit" phrase or a dictation segment.

## Limitations and Edge Cases

**`journalMode` is a live binding caveat** — ES module `export let` creates a live binding that
is readable from other modules, but in `app.js` you reference `journalMode` from the import. If
`app.js` is not a module (Option B from `WEATHER.md` — inline functions), use a getter function
instead: `export function isJournalMode() { return journalMode; }` and call
`isJournalMode()` in `app.js`.

**LLM generates summary in the same voice** — Because the summary call uses `ephemeralMessages`
with `SYSTEM_PROMPT`, the LLM will write the summary in Starling's voice but **not** speak it
aloud automatically (it is rendered to the review panel only). The `sendToOllama` return value
is the raw summary string — `submitJournalEntry` uses the return value directly rather than
relying on the normal TTS pipeline.

**Very long entries** — If the user dictates 20+ segments, the raw transcript may approach
4 000–6 000 tokens. This is within all supported models' context windows. The LLM prompt
asks for a 2–4 sentence summary so the output is always concise regardless of input length.

**Offline / backend down** — The `confirmJournalEntry()` fetch is wrapped in try/catch.
On failure the user hears "Could not save the entry" and the review panel stays open so
they can retry. The raw transcript is preserved in `_pendingSave` until they discard or navigate away.
